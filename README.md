## **SOBRE O PROJETO**

Este repositório contém o código-fonte do meu portfólio profissional online. Ele foi projetado para demonstrar minhas habilidades de desenvolvimento Front-End e Back-End, reunindo minhas principais experiências na área de TI, meus projetos práticos e minhas certificações acadêmicas de forma interativa, fluida e com alto valor estético.

---

## **PRINCIPAIS RECURSOS DO PORTFÓLIO**

*   **Design Premium (Glassmorphism)**: Interface moderna desenvolvida com um sistema visual baseado em transparências, sombras suaves, gradientes em tons roxos e elementos dinâmicos.
*   **Tema Claro & Escuro (Dark/Light Mode)**: Alternador de tema integrado que salva a preferência do usuário diretamente no armazenamento local do navegador (`localStorage`).
*   **Carrossel Automático Inteligente**: Os slides de projetos e certificados passam sozinhos em intervalos ideais de leitura, com um sistema de pausamento inteligente ao passar o mouse (`hover`) ou ao tocar na tela (`touch`).
*   **Visualizador de Imagens Contextual (Lightbox)**: Ao clicar em qualquer print de tela dos projetos ou certificados, a imagem abre em destaque em tela cheia com desfoque de fundo (`backdrop blur`). O visualizador agrupa as imagens por contexto, permitindo passar para o próximo print usando cliques nas setas laterais ou as setas físicas do teclado (`ArrowLeft` / `ArrowRight`).
*   **Design 100% Responsivo**: Layout otimizado para proporcionar uma ótima experiência de navegação em celulares, tablets e computadores.

## **COMO EXECUTAR LOCALMENTE**

1. Clone o repositório em sua máquina local:
   ```bash
   git clone https://github.com/FelipeBadoco/Portfolio.git
   ```

2. Acesse a pasta do projeto:
   ```bash
   cd Portfolio
   ```

3. Abra o arquivo `index.html` diretamente em seu navegador web de preferência ou execute um servidor local (como a extensão Live Server do VS Code).
